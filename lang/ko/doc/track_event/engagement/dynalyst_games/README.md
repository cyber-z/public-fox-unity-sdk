| 파라메터명 | 타입 | 필수 | 설명 |
| ------------- | :-------------: | :-------------: | -------------:|
| guid | String | | 게임 유저 ID |
| adnt | int | | 앱 opt out <br>opt out의 경우에는 1、지정하지 않는 경우에는 0 |
| ext | JsonObject | | 유저 데이터<br>상세는 Dynalyst사에 문의하여 주십시오 |
